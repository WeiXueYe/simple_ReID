# 视频镜头切换ID分裂问题解决方案

## 🎯 问题描述

视频中存在镜头切换时，同一个人被识别为多个ID的问题，特别是在黑夜环境和有水印干扰的情况下。

### 2. 动态相似度阈值调整

**实现位置**: `src/person_tracker.py` - `_get_current_similarity_threshold()`

**策略**:
- 正常状态: 0.75 (严格匹配)
- 镜头切换后: 0.68 (宽松匹配)
- 30帧内逐步恢复

**优势**: 适应镜头切换后的外观变化，减少ID分裂

### 3. 动态特征更新权重

**实现位置**: `src/person_tracker.py` - `_get_current_feature_weight()`

**策略**:
- 正常状态: 15% 更新权重 (保守更新)
- 镜头切换后: 35% 更新权重 (快速适应)

**优势**: 快速适应新镜头下的人物外观变化

### 4. 增强的ID合并机制

**实现位置**: `src/person_tracker.py` - `_emergency_merge_similar_persons()`

**功能**:
- 镜头切换时自动触发紧急合并
- 合并阈值: 0.70 (比正常0.85更宽松)
- 时间重叠验证确保合并准确性

### 5. 轨迹历史管理

**实现位置**: `src/person_tracker.py` - `person_histories`

**功能**:
- 维护每个人物50帧的轨迹历史
- 支持位置预测和轨迹分析
- 辅助判断身份一致性

### 6. 配置参数优化

**实现位置**: `src/config.py`

**新增参数**:

## 🎯 针对您具体问题的解决方案

### 问题1: "1.mp4中三人识别成多人"

**根本原因**: 镜头切换导致特征变化，同一人物被分配不同ID

**解决方案**:
1. ✅ 镜头切换自动检测
2. ✅ 切换后使用更宽松的匹配阈值
3. ✅ 紧急ID合并机制
4. ✅ 轨迹历史辅助判断

### 问题2: "黑夜环境影响识别"

**解决方案**:
1. ✅ 升级到 pcb_p6 高精度模型
2. ✅ 提高输入分辨率 (1280x1280 YOLO, 256x512 ReID)
3. ✅ 动态特征权重适应光照变化
4. ✅ 增强的相似度计算

### 问题3: "水印干扰"

**解决方案**:
1. ✅ 镜头切换检测可识别水印出现/消失
2. ✅ 时间连续性机制减少瞬时干扰影响
3. ✅ 多帧特征融合提高稳定性
